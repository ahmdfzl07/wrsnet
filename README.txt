Syarat & Ketentuan Penggunaan Source Code Aplikasi

1. Lisensi Penggunaan Internal, Source code FLAYNET Dashboard hanya untuk penggunaan internal pembeli. DILARANG menjual ulang (resale), menyewakan, atau mendistribusikan ulang ke pihak ketiga.

2. Garansi & Dukungan, Garansi bugfix selama 30 hari sejak source code diserahkan. Dukungan teknis (bantuan setup, koneksi MikroTik & GenieACS) selama 90 hari pertama via WhatsApp pada jam kerja (Senin–Jumat 09.00–19.00 WIB).

3. Modifikasi, Pembeli berhak memodifikasi source code untuk kebutuhan internal, termasuk rebranding (logo, nama, warna). Modifikasi tidak menghilangkan kewajiban di poin 1.

4. Hak Cipta, Hak cipta source code tetap milik penjual. Pembeli memperoleh lisensi lifetime penggunaan tanpa enkripsi apapun.

5. Pembayaran, Pembayaran dilakukan via transfer bank ke rekening yang tertera. Untuk proses pembayaran pembeli wajib mentransfer sejumlah harga yang telah ditentukan 

6. Tanpa Refund, Setelah source code diserahkan dan email ini terkirim, tidak ada pengembalian dana dengan alasan apapun.

7. After Sales, Kami menyediakan layanan after sales, jika terjadi kendala dapat menghubungi kami melalui WhatsApp untuk dibantu lebih lanjut selama masa garansi dan di jam kerja.
